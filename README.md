# 専攻科 口頭試問対策（4力フラッシュカード）

機械工学専攻の学生が、専攻科進学の口頭試問で問われる「4力（材料・機械・熱・流体）」の基礎概念を効率よく暗記するためのWebアプリです。

## 🚀 技術スタック
- **Framework**: React 18 (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide-react

## 🛠 セットアップ
1. プロジェクトを作成
   ```bash
   npm create vite@latest kosen-quiz-app -- --template react-ts
   cd kosen-quiz-app
